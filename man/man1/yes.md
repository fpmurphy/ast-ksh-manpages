# NAME

yes - repeatedly out a string until killed

# SYNOPSIS

`yes` \[ `options` \] \[string\]

# DESCRIPTION

`yes` repeatedly writes `string` to standard output, followed by a
`newline` character, until it is terminated. If `string` is omitted then
then `y` is used.

# EXIT STATUS

`0`
: Success.

`&gt;0`
:   An error occurred.
